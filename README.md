# Proyecto_Asistencia_S-S
Este script fue desarrollado en VS Code con formato “.py"(python)
Es un Registro de asistencia simple el cual llamé “Registrador S-S”, el cual sirve para ingresar nombre y apellido junto con su asistencia (presente, atrasado, ausente).
Este programa permite:
1)	Ingresar nombre, apellido: Solicita nombre y apellido y luego su asistencia.
2)	Mostrar asistencia: Muestra todos los usuarios que han sido ingresados previamente al sistema.
3)	Eliminar asistencia: Borra 1 registro a elección del usuario.
4)	Salir: Cierra el programa
